# 第一篇：关于这个小站

这是一篇示例文章，用来演示阅览页的排版效果。

## 支持的格式

- 标题（h1 / h2 / h3）
- 列表
- **加粗**、*斜体*
- `行内代码`
- 引用

> 把 `doc/` 目录下的 md 文件换成你自己的内容，再更新 `articles.json` 里的条目即可。

```
console.log("代码块也没问题");
```

英文段落 The quick brown fox jumps over the lazy dog. 用来测试英文字体（./font/mono.ttf）与中文字体混排的效果。
